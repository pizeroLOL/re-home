# re-home

一个尝试复刻 [nanorocky/home](https://github.com/nanorocky/home) 的产物

## 使用方法

1. 把图片放入 `public/backgound-image.png` 里
2. `bun install`
3. `bun gen`
4. `bun run build`
5. 把 `dist` 随便扔到一个服务器上跑就行
